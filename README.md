# Тестовое задание на стажировку JetBrains
Создание системы для перевода текстов на платформе Stepik

## Как запустить
```
# Установить Docker и docker-compose
docker-compose up -d
```

## Как проверить работу
```
curl localhost:8000/lessons/18630
```

## Скорость сервиса
В папке benchmarks.

- Скорость сервиса: 2.16 rps
- Без подключения к Stepik API: 1596.35 rps
- Без кеширования: 0.26 rps
